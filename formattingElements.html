HTML Formatting Elements

HTML formatting is a tool in HTML through which we can add some importance or we can say increase the importance of particular texts by using various tags. Each tag has different properties.

HTML Formatting Elements are as follow:

  • < b > tag

    • This tag is used to make any text bold without any importance.

    For Example:
    This text is bold.

  • < em > tag

    • This tag is used to make any text emphasised.

    For Example:
    This text is emphasised.

  • < i > tag

    • This tag is used to make any text italic.

    For Example:
    This text is italic.

  • < ins > tag

    • This tag is used to show the viewer that the particular text has been added later in the content. That text will be underlined.

    For Example:
    This text is newly inserted.

  • < mark > tag

    • This tag is used to make any text highlighted in the paragraph.

    For Example:
    This is an important word, so it is highlighted using the mark tag.

  • < sub > tag

    • This tag is used to make subscript of any text.

    For Example:
    This text is written in subscript form.

  • < sup > tag

    • This tag is used to make superscipt of any text.

    For Example:
    This text is written in superscript form.

  • < del > tag

    • This tag is used to show the viewer that the particular text has been deleted from the content. There will be a line passing through the middle of that particular text.

    For Example:
    This text is deleted from the content using the delete tag.

  • < strong > tag

    • This tag is used to show the importance of any particular text. It makes the text bold.

    For Example:
    The given Statement is termed as strong so it is in bold using the strong tag.

  • < small > tag

    • This tag is used to make any text smaller in size as compared to other texts in the paragraph.

    For Example:
    The word big is actually smaller than others.
