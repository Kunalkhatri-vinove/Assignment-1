Lists in HTML

With the help of list tag in html we can add multiple items under these section.
We use < li > tag before any item to mark it as a list.

UNORDERED LIST

An unorderd list is type of list in html in which there is no numbering and there is no relevance of priority. In unordered list we use < ul > tag.

In unorderd list items are marked with bullets.

Example

Some of the diary Products are as folows

  • Milk
  • Curd
  • Butter
  • Cheese
  • Paneer
  • Dried Milk

ORDERED LIST

An orderd list is type of list in html in which there is numbering and we create to arrange items in certain order. In ordered list we use < ol > tag.

In orderd list items are marked with their numbering.

Example

SEVEN WONDERS IN THE WORLD

  1. The Colosseum in Italy
  2. Petra in Jordan
  3. Chichén Itzá in Mexico
  4. Christ the Redeemer in Brazil
  5. Machu Picchu in Peru
  6. Taj Mahal in India
  7. The Great Wall of China

DESCRIPTION LIST

In description list, there are no of items in the list, and there is a description about each term present in the list

In description list there is no any numbering or bullets.

There are various tags used in the description list.
< dl > tag is used to define the description list.
< dt > tag is used to define the term in the list.
< dd > tag is used to describe the term in the list.

Example
